PHASE 2: CREATE UTILITY MODULES

Follow these steps to implement the utility modules:

1. Open and edit frontend\js\utils\api.js:
   - Define API_BASE_URL and API_ENDPOINTS
   - Create fetchApi function for making API calls
   - Implement specific API call functions
   - Make sure to export all functions and constants

2. Open and edit frontend\js\utils\state.js:
   - Create appState object with data tracking properties
   - Add methods to update state
   - Add session storage integration
   - Export the state object

3. Open and edit frontend\js\utils\ui.js:
   - Implement showError function
   - Implement showLoading function
   - Implement activateTab function
   - Add other UI helper functions
   - Export all functions

4. Open and edit frontend\js\utils\formatters.js:
   - Implement formatDate function
   - Implement formatNumber function
   - Implement formatMetricName function
   - Implement formatMetricValue function
   - Implement formatParamName function
   - Export all functions

Use the Function Migration Checklist in the modularization-plan.md 
to make sure you've moved all the necessary functions.

After completing these steps, run the module.bat file again to move to the next phase.
